# 🍅 番茄鐘 Pomodoro Timer

一個具有日式動漫風格的番茄鐘網頁應用程式，幫助你更好地管理時間和提升專注力。

## 🌸 線上 Demo

立即體驗：[https://stanley-hsieh-1219.github.io/Pomodoro-Technique/](https://stanley-hsieh-1219.github.io/Pomodoro-Technique/)

## ✨ 功能特色

### 核心功能
- ⏱️ **可自訂計時時長**：提供 5、10、15、20、30 分鐘選項
- 🕐 **即時時間顯示**：顯示當前日期與時間（西元年月日 HH:MM:SS 格式）
- ▶️ **完整計時控制**：開始、暫停、重置功能
- 📊 **視覺化進度條**：即時顯示剩餘時間進度
- 🔔 **多重提醒機制**：
  - 音效提醒（自動播放提示音）
  - 瀏覽器通知提醒

### 視覺設計
- 🎨 **日式動漫風格**：採用粉紅與紫色漸層主題
- 🌸 **櫻花動畫效果**：營造輕鬆愉悅的使用氛圍
- ✨ **發光特效動畫**：增添視覺趣味
- 📱 **響應式設計**：支援桌面與行動裝置

## 📖 操作說明

1. **選擇時間**：點擊上方的時間按鈕（5分、10分、15分、20分、30分）選擇計時長度
2. **開始計時**：點擊「開始」按鈕啟動計時器
3. **暫停計時**：需要暫停時點擊「暫停」按鈕
4. **重置計時**：點擊「重置」按鈕將計時器恢復到初始狀態
5. **查看進度**：底部進度條會即時顯示剩餘時間百分比
6. **時間到提醒**：計時結束時會播放提示音並顯示瀏覽器通知

## 🎯 番茄工作法介紹

番茄工作法（Pomodoro Technique）是一種時間管理方法，由 Francesco Cirillo 在 1980 年代末期創立。

### 基本步驟：
1. 決定待完成的任務
2. 設定番茄鐘（傳統為 25 分鐘）
3. 專注工作，直到番茄鐘響起
4. 短暫休息 3-5 分鐘
5. 每完成 4 個番茄鐘後，休息 15-30 分鐘
